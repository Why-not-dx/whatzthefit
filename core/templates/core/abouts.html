About

Hi, I'm Anthony, I made this website.
It's nothing crazy, but it was fun enough for me to learn how to make a website using Python.

Whatzthefit is a way for you to share and compare how popular clothing might fit on you. For now, the information provided is limited but might expand as time goes.

Projects on stand by are :

  • Adding information about the users to make the grading system more meaningful by considering factors like height, weight, or other data.
  • More information about the items is also interesting. Is it see-through? Very short? Maybe it's itchy?

Privacy

This website was designed as an exercise to learn web development. No information about the users will be used or shared in any way.

Terms of use

Whatzthefit is purrely informative and takes users input for it's sizing grade.
We don't have any team or resource to check on the veracity of the grades. I guess you need to trust the majority !

Any user uploading wrong informations, wrong pictures, or anything unrelated to clothing and apparels will be banned and their posts deleted. Any self promotion or link is prohibited.

The grades can be used on your own projects as long as you credit Whatzthefit. An API might be available soon.

NUDITY

Nudity in itself is not an issue if the piece of clothing requires it. Models bodies are not considered an offense.
But unnecessary nudity will be sanctionned. Nudity will be considered unnecessary if the crotch is not covered while the piece of clothing should cover it or if the item is unrelated to those body parts. Breasts are not considered offensive but should never be central to the picture if the item is unrelated to the chest.
